# GitHub Webhook

## Quick Start

```bash
# Webhook Secret setzen
export GITHUB_WEBHOOK_SECRET=your-secret
```

## Unterstützte Events

- `issues` (opened) → AI intake
- `push` → Deploy trigger
- `pull_request` (opened) → Review
